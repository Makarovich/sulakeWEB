Welcome to the sulakeWEB Github Repository

About sulakeWEB

sulakeWEB was created for the sole purpose of bringing some relevance back to the Habbo Hotel Development community. It is the only CMS built for habbo to primarily use a bunch of javascript, ajax, jquery, etc. code. It doesn't use any of Sulake's styling, because it sports a sleek black and white style created by the main developer himself. sulakeWEB is still in development but pretty close to a release to watch out!

Authors and Contributors

sulakeWEB was created by Cobe 'Makarov'. It started out a small system but with the increasing knowledge and the developments of his side-project frameworks, sulakeWEB is flourished into a top-of-the-line, modern content management system.
